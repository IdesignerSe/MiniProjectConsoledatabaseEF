# MiniProjectConsoledatabaseEF

Proyecto de consola en C# utilizando Entity Framework Core para gestionar:
- Students
- Courses
- Enrollments
- StudentProfile

Incluye:
- CRUD completo
- Relaciones One-to-One, One-to-Many y Many-to-Many
- Data Seeding
- Migraciones con EF Core

## Requisitos
- .NET 8 SDK
- SQL Server LocalDB

## Ejecutar el proyecto
```bash
dotnet run --project ConsoleApp/ConsoleApp.csproj
